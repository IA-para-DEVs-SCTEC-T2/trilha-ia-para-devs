# Exemplos de Testes Automatizados

Referencia pratica de testes unitarios, de API e E2E.

## Arquivos disponíveis

| Arquivo | Descricao |
|---------|-----------|
| `exemplo-teste-unitario-python.md` | Testes unitarios com pytest |
| `exemplo-teste-api-postman.md` | Testes de API com Postman |
| `exemplo-casos-de-teste.md` | Estrutura de casos de teste |
| `boas-praticas-testes.md` | Boas praticas gerais de testes |

## Tipos de teste

| Tipo | O que testa | Ferramenta |
|------|-------------|------------|
| Unitario | Funcoes e metodos isolados | pytest, Jest |
| API | Endpoints HTTP | Postman, pytest + requests |
| E2E | Fluxos completos no navegador | Playwright, Cypress |
