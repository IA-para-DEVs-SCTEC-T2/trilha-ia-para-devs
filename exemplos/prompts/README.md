# Exemplos de Prompts

Prompts reutilizaveis para tarefas comuns de desenvolvimento de software assistido por IA.

## Arquivos disponíveis

| Arquivo | Descricao |
|---------|-----------|
| `prompt-gerar-prd.md` | Prompt para gerar Product Requirements Document |
| `prompt-gerar-user-stories.md` | Prompt para gerar User Stories com BDD |
| `prompt-criar-diagrama-uml.md` | Prompt para criar diagramas UML |
| `prompt-criar-readme.md` | Prompt para gerar README de projeto |
| `prompt-revisar-codigo.md` | Prompt para revisao de codigo |
| `prompt-gerar-testes.md` | Prompt para gerar testes automatizados |
| `prompt-documentacao-tecnica.md` | Prompt para documentacao tecnica de modulos |
| `prompt-gerar-issues-github.md` | Prompt para gerar Issues no formato GitHub |

## Como usar

Cada arquivo contem um prompt completo com contexto, objetivo, entrada esperada, restricoes e formato de saida. Adapte o contexto para o seu projeto antes de usar.

## Importante

Sempre revise o resultado gerado pela IA antes de usar. Os prompts sao pontos de partida, nao garantias de resultado perfeito.
