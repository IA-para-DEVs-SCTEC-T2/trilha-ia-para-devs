# Exemplos de GitHub

Referencia pratica de Issues, Pull Requests, commits, branches e fluxo de trabalho com GitHub.

## Arquivos disponíveis

| Arquivo | Descricao |
|---------|-----------|
| `exemplo-issue.md` | Modelo de Issue com User Story e criterios de aceitacao |
| `exemplo-pull-request.md` | Modelo de Pull Request com descricao e checklist |
| `exemplo-commit-messages.md` | Convencoes de mensagens de commit |
| `exemplo-github-flow.md` | Fluxo basico de desenvolvimento com GitHub |
| `exemplo-branches.md` | Convencoes de nomes de branches |

## Fluxo resumido

```
Issue aberta -> Branch criada -> Commits -> Push -> Pull Request -> Revisao -> Merge
```

Consulte `exemplo-github-flow.md` para o fluxo detalhado.
