# Semana 02 - Engenharia de Contexto

## Objetivo da semana

Apresentar os fundamentos de engenharia de contexto e prompts, ensinando como estruturar instrucoes para obter respostas mais precisas e uteis de modelos de linguagem.

## Conteudos principais

- O que e engenharia de contexto
- Diferenca entre prompt simples e prompt estruturado
- Tecnicas de prompting: zero-shot, few-shot, chain-of-thought
- Como fornecer contexto adequado para a IA
- Limitacoes dos modelos e como contorna-las
- Aplicacoes praticas no desenvolvimento de software

## Como estudar o material

1. Leia os slides antes ou apos a aula para revisar os conceitos.

4. Consulte `materiais/guias/guia-uso-de-ia-no-desenvolvimento.md` para aprofundamento.

## Relacao com o mini-projeto

A engenharia de contexto e a base para todas as interacoes com IA ao longo do modulo. No mini-projeto, voce usara essas tecnicas para gerar PRDs, User Stories, testes e documentacao tecnica. Registre os prompts utilizados para compor o historico de uso de IA exigido na entrega.
