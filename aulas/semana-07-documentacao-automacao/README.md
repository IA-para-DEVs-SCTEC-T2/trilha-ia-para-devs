# Semana 07 - Documentacao e Automacao

## Objetivo da semana

Apresentar como usar IA para produzir documentacao tecnica de qualidade e automatizar tarefas repetitivas no fluxo de desenvolvimento, como geracao de READMEs, changelogs, relatorios e scripts de automacao.

## Conteudos principais

- Documentacao tecnica assistida por IA
- Geracao de READMEs, changelogs e relatorios
- Automacao de tarefas com scripts e GitHub Actions
- Boas praticas de documentacao em projetos de software
- Integracao entre documentacao e fluxo de desenvolvimento

## Relacao com o mini-projeto

A documentacao e um dos criterios de avaliacao do mini-projeto. Use o aprendizado desta semana para:

- Escrever um README claro e completo para o seu projeto.
- Documentar decisoes tecnicas e arquiteturais.
- Automatizar tarefas repetitivas do seu fluxo de desenvolvimento.
