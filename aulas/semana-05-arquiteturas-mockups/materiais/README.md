# Materiais - Semana 05

| Arquivo | Descricao |

## Sobre o manual

O manual disponivel nesta pasta orienta um fluxo completo de engenharia de software assistida por IA usando Kiro, GitHub, Issues, Projects, Pull Requests, automacao e documentacao.

Ele cobre:

- Como usar o Kiro como assistente de desenvolvimento
- Como organizar um repositorio GitHub para um projeto de software
- Como criar e gerenciar Issues e Projects
- Como estruturar Pull Requests e revisoes de codigo
- Como automatizar tarefas com scripts e GitHub Actions
- Como gerar e manter documentacao tecnica com apoio de IA

## Como usar

Leia o manual como complemento aos slides da aula. Ele serve como referencia durante o desenvolvimento do mini-projeto e das atividades semanais.
