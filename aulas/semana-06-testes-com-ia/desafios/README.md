# Desafios - Semana 06

Esta pasta contem os desafios praticos da Semana 06, focados em testes automatizados com apoio de IA.

## Desafios disponíveis

| Pasta | Descricao |
|-------|-----------|
| `01-testes-e2e-login/` | Desafio de testes E2E para fluxo de login |

## O que os alunos devem fazer

1. Leia as orientacoes gerais em `orientacoes-gerais.md`.
2. Acesse a pasta do desafio escolhido.
3. Leia o enunciado e entenda o codigo base disponivel.
4. Crie, revise ou corrija os testes conforme solicitado.
5. Execute os testes e registre as evidencias.
6. Siga as instrucoes de entrega em `como-entregar.md`.

## Criterios de qualidade

Consulte `criterios-de-qualidade.md` para os criterios esperados em cada entrega.
