# Semana 06 - Testes com IA

## Objetivo da semana

Apresentar como usar IA para criar, revisar e corrigir testes automatizados, cobrindo testes unitarios, testes de API e testes E2E.

## Conteudos principais

- Tipos de testes: unitario, de API e E2E
- Escrita de testes com pytest
- Validacao de APIs com Postman
- Testes E2E com Playwright
- Uso de IA para geracao e revisao de testes
- Cobertura de testes e analise de riscos
- Cenarios positivos, negativos, limites e edge cases

## Como estudar o material

1. Leia os slides para entender os conceitos de cada tipo de teste.
2. Acesse `hands-on/` e leia o README de cada subpasta.
3. Execute os exemplos localmente para ver os testes em acao.
4. Leia as orientacoes em `desafios/` antes de comecar o desafio.
5. Consulte `exemplos/testes/` para referencia adicional.

## Relacao com o mini-projeto

No mini-projeto, voce deve incluir testes automatizados para as funcionalidades implementadas. Use os exemplos desta semana como referencia e consulte `materiais/guias/guia-testes.md` para boas praticas.
