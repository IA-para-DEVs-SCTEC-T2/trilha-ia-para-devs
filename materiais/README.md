# Materiais de Apoio

Esta pasta reune materiais gerais de apoio ao modulo, nao vinculados a uma unica aula ou semana.

## Estrutura

| Pasta | Conteudo |
|-------|----------|
| `guias/` | Guias praticos de referencia rapida |
| `templates/` | Templates copiáveis e preenchíveis |
| `referencias/` | Listas de referencias por tema |
| `ebooks/` | Sugestoes de leitura e materiais complementares |

## Diferenca entre as subpastas

- `guias/`: explicam como usar uma ferramenta ou tecnica. Leia quando precisar aprender ou revisar um conceito.
- `templates/`: modelos prontos para preencher. Use como ponto de partida para artefatos do projeto.
- `referencias/`: listas de links e materiais externos organizados por tema. Consulte para aprofundamento.
- `ebooks/`: sugestoes de livros, apostilas e materiais de leitura complementar.

## Quando consultar esta pasta

- Ao iniciar uma atividade e precisar de um template.
- Ao precisar revisar um conceito de Git, testes ou documentacao.
- Ao buscar referencias para aprofundar um tema do modulo.
- Ao preparar artefatos do mini-projeto.
