# Templates

Templates copiáveis e preenchíveis para os principais artefatos do modulo.

## Arquivos disponíveis

| Arquivo | Uso |
|---------|-----|
| `template-prd.md` | Product Requirements Document |
| `template-user-story.md` | User Story com BDD |
| `template-issue.md` | Issue no GitHub |
| `template-pull-request.md` | Pull Request |
| `template-readme-projeto.md` | README de projeto |
| `template-caso-de-teste.md` | Caso de teste |
| `template-relatorio-entrega.md` | Relatorio de entrega |

## Como usar

Copie o arquivo do template para a pasta do seu projeto e preencha os campos marcados com `[...]`.
