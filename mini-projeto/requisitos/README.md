# Requisitos

Esta pasta contem os artefatos de requisitos do mini-projeto.

## Arquivos disponíveis

| Arquivo | Descricao |
|---------|-----------|
| `prd.md` | Product Requirements Document completo |
| `requisitos.md` | Tabela de requisitos do projeto |
| `regras-de-negocio.md` | Regras que o sistema deve respeitar |
| `escopo.md` | O que esta dentro e fora do escopo |

## Como usar

Preencha os arquivos com as informacoes do seu projeto. Use os exemplos do Shop4u apenas como referencia didatica, nao como modelo obrigatorio.

Comece pelo `prd.md` e use os demais arquivos para detalhar cada secao.
