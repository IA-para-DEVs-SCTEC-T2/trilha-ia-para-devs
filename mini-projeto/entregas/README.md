# Entregas

Esta pasta contem os modelos para organizar e registrar a entrega do mini-projeto.

## Arquivos disponíveis

| Arquivo | Descricao |
|---------|-----------|
| `checklist-entrega.md` | Checklist final antes de entregar |
| `evidencias.md` | Modelo para registrar evidencias |
| `links.md` | Modelo para listar links importantes |
| `apresentacao-final.md` | Roteiro para a apresentacao tecnica |

## Como usar

Preencha cada arquivo com as informacoes do seu projeto antes da entrega. O `checklist-entrega.md` deve ser o ultimo a ser verificado.
